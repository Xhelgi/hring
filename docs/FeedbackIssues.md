# Feedback & Issues

If you find a bug or have a suggestion:

1. **Check existing [Issues](https://github.com/Xhelgi/hring/issues):** Someone might already be working on it, or you might find a discussion you can join. 
2. **Contribute:** If you see an open issue that you’d like to fix yourself, feel free to pick it up! We improve it together.
3. **Open a new Issue:** If your bug or idea isn't listed yet, please open a new one. (you can also write that on the [Discussions Page](https://github.com/Xhelgi/hring/discussions)).

Please use the following **labels** for your reports:
- `bug`: Something isn't working.
- `enhancement`: New features.
- `ui`: UI improvements.
- `refactor`: Code quality and structural changes.
- `documentation`: Fixes or additions to the README and docs.
