# Security Policy

## Supported Versions
Only the latest version of `hring` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it to me by emailing `helgix@proton.me`.

Please include the following details with your report:
- A description of the vulnerability.
- Steps to reproduce it.

I will do my best to respond to your report within 48 hours.
