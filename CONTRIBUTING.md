# Contributing to Hring

First off, thank you for considering contributing. It's people like you that make open source such a great community!

## How Can I Contribute?
### Reporting Bugs
If you find a bug, please open an issue and provide as much detail as possible, including:
- Your OS and Window Manager
- Steps to reproduce the bug
- Any relevant log output

### Suggesting Enhancements
Feature requests are welcome. Open an issue and describe the feature you'd like to see.

### Pull Requests
If you want to contribute code, please follow these steps:
1. Fork the repository.
2. Create a new branch (`git switch -c feature/AmazingFeature`).
3. Make your changes.
4. Ensure your code is formatted with `cargo fmt`.
5. Open a Pull Request.
